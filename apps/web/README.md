# Web App

This app will host the ModelSentinel dashboard.

Planned component areas:

- `src/components` for cards, tables, panels, and controls
- `src/charts` for distribution and PSI trend charts
- `src/lib` for API clients and formatters
- `src/mocks` for temporary frontend demo data during Phase 1

Recommended first files:

- `src/App.tsx`
- `src/components/FeatureTable.tsx`
- `src/components/SummaryCards.tsx`
- `src/charts/DistributionChart.tsx`
- `src/charts/PsiTrendChart.tsx`
